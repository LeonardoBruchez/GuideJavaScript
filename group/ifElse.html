If e else

Exemplos de If:

  • var vel = 70
  • console.log(`A velocidade do seu carro é de ${vel}Km/h`)
  • if (vel > 60) { // Condição simples
  • console.log(`Você ultrapassou o limite de velocidade. Multado`)
  • }
  • console.log(`Dirija sempre usando cinto de segurança!`)
  • Usar extensão code Runner + CTRL + ALT + N

Exemplos de else:

  • var pais = 'Brasil'
  • console.log(`Vivendo em ${pais}`)
  • if (pais == 'Brasil' || pais == 'brasil' || pais == 'brazil' || pais == 'Brazil') {
  • console.log('Brasileiro')
  • } else {
  • console.log('Estrangeiro')
  • }
  • Usar extensão code Runner + CTRL + ALT + N

"Pais" for diferente

  • var pais = 'UK'
  • console.log(`Vivendo em ${pais}`)
  • if (pais != 'Brasil' && pais != 'brasil' && pais != 'brazil' && pais != 'Brazil') {
  • console.log('Você é Estrangeiro')
  • } else {
  • console.log('Você é Brasileiro')
  • }
  • Usar extensão code Runner + CTRL + ALT + N
  • && necessite que todos sejam diferentes,nesse caso se um for igual ao "pais" então será Brasileiro,se todos forem diferente da variável pais então será diferente
